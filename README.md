# 💳 UPI Transaction Dashboard

An interactive **Power BI Dashboard** designed to analyze and visualize UPI (Unified Payments Interface) transactions across multiple dimensions, including banks, cities, payment methods, demographics, merchants, and transaction types.

The dashboard enables users to uncover transaction patterns, monitor balances, and gain valuable insights into digital payment behavior through interactive visualizations and filters.

---

## 📊 Dashboard Overview

This Power BI dashboard provides:

* Monthly balance trend analysis
* Transaction amount monitoring
* Bank-wise transaction insights
* Merchant performance analysis
* Customer demographic analysis
* Payment method distribution
* Interactive filtering for deeper exploration

---

## ✨ Key Features

### 🔍 Interactive Filters

The dashboard includes the following slicers for detailed analysis:

* Bank Name Sent
* Bank Name Received
* City
* Device Type
* Gender
* Age Group
* Merchant Name
* Payment Method
* Purpose
* Transaction Type

Users can dynamically filter the entire dashboard based on these dimensions.

---

## 📈 Visualizations

### Balance by Month

* Tracks monthly account balance trends throughout the year.
* Helps identify seasonal fluctuations and transaction patterns.
* Available in both:

  * Line Chart
  * Column Chart

### Transaction Analysis

* Monitor transaction amounts over time.
* Compare transaction performance across months.

### Demographic Insights

Analyze transaction behavior by:

* Gender
* Age Group
* Device Type

### Banking Insights

* Compare sending and receiving banks.
* Analyze fund flow between financial institutions.

### Merchant & Purpose Analysis

* Identify top merchants.
* Understand common transaction purposes.
* Track spending behavior patterns.
